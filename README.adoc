= Spring AI Demo

A simple demo of common ai agent concepts

== Running

You will need `Docker` installed to run the LLMS

Execute the below command to start the ollama api server.

```
docker run \
 --rm \
 -e OLLAMA_DEBUG=2 \
 -v ~/ollama_models:/root/.ollama \
 -p 11434:11434 \
 --name ollama \
 ollama/ollama
```

You will need three models to run the examples - in a terminal run the below commands to fetch the LLMs

```
# Normal Chat
docker exec -it ollama ollama pull llama3.1

# Vision
docker exec -it ollama ollama pull gemma3

# Rag
docker exec -it ollama ollama pull mxbai-embed-large
```

Once ollama is running you can run our local spring API using `bootRun`

```
./gradlew bootRun
```
